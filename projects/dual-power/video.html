This stylized simulation visualizes an agent (represented by a knight) making effort-reward decisions while navigating a procedural dungeon. Using a subjective value model, the agent evaluates adjacent rooms, balancing reward potential (coin icons) against challenge rating (sword icons). The agent expends energy to overcome room challenges and collect treasure as rewards (score), with the calculated subjective values ("V") displayed during decision phases. The visualization displays the decision-making process as the agent progresses through increasingly challenging environments, demonstrating optimization of resource-constrained choices.
