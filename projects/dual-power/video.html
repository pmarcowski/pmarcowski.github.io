Overview
This stylized simulation visualizes an agent (represented by a knight) making effort-reward decisions while navigating a procedural dungeon. Using a dual-system power model, the agent calculates subjective value (v) for each room option according to: v(x) = x × [1 + ωδ₁E^γ₁ - (1-ω)δ₂E^γ₂], where x is the nominal reward value (coins), E is effort-challenge level (swords), and other parameters represent individual effort preferences.

Decision Process
The agent evaluates adjacent rooms, balancing reward potential against required effort, with calculated subjective values (v) displayed during decision phases. For example, when facing a room with 5 coins requiring 3 swords of effort, the agent calculates the positive contribution from effort (potentially enhancing value) and subtracts the negative contribution (effort cost), resulting in a final subjective value that may be higher or lower than the nominal value.

Environment
Room outlines are color-coded to indicate challenge progression: green for low challenge, yellow for medium, and red for high challenge.

Resources
As the agent clears rooms, it expends energy from its limited energy pool to overcome challenges. The agent's accumulated rewards (coins collected) are displayed as the Score, representing overall success.

Insights
The visualization demonstrates how effort can have positive, negative, or nonmonotonic impacts on value as the agent optimizes resource-constrained choices.
